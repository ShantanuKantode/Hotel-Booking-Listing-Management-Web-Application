# Hotel Booking & Listing Management Web Application

A full-stack property booking and listing management platform inspired by Airbnb. 
The application is being developed using Node.js and Express with server-side rendered views.

## Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- EJS (Server-side rendering)
- Bootstrap (UI)
- RESTful APIs
- MVC Architecture

## Key Features
- User authentication and authorization (in progress)
- Property listing with full CRUD operations
- Dynamic routing and server-side rendering using EJS
- MongoDB schema design with validation
- Admin-level listing management (planned)

## Security (Learning & Implementation)
- Exploring secure authentication using Passport.js
- Understanding password hashing & salting concepts (PBKDF2)

## Project Status
🚧 Ongoing — core backend functionality implemented.  
Additional features like authentication, maps, and deployment are under development.

## How to Run Locally
1. Clone the repository
2. Run `npm install`
3. Start MongoDB locally
4. Run `node app.js`
